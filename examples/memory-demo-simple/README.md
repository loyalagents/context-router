# Memory Demo Simple

This is a loose scratch area for hand-authored memory demo ideas.

Use this folder when you want to quickly dump a user, form, or scenario idea without learning the structured fixture contract in `examples/memory-demo/`.

Rules:

- Keep all data synthetic and non-sensitive.
- Use plain Markdown.
- Do not expect anything in this folder to be validated or runnable.
- When an idea needs to become a real demo scenario, promote it into `examples/memory-demo/`.

Suggested layout:

- `users/`: rough user memory dumps.
- `forms/`: rough form dumps and field requirements.

The structured demo folder remains the source of truth for verified scenarios.
