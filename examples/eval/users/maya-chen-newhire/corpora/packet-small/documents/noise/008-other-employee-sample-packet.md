# Sample Payroll Packet

This training packet shows how a completed payroll packet can look for a
different employee. The sample employee is Jordan Avery, and the record is kept
only as a payroll operations example. It is not attached to Maya Chen's worker
id and should not supply values for Maya's forms.

Sample sections shown in this packet include a withholding setup page, a direct
deposit page, and a short reviewer checklist. Values in those sections are
demo-style values for the sample worker and are not employee evidence for any
current onboarding packet.
